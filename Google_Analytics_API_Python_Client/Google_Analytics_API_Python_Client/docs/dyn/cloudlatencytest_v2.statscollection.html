Args:
  body: object, The request body. (required)
    The object takes the form of:

{
    "stats": [
      {
          "stringValues": [
            {
              "value": "A String",
              "label": "A String",
            },
          ],
          "intValues": [

              "label": "A String",
            },
          ],
          "doubleValues": [
            {
              "value": 3.14,
              "label": "A String",
            },
          ],
          "time": 3.14,
        },
    ],
  }


Returns:
  An object of the form:

    {
    "testValue": "A String",
  }
